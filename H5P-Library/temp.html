Quick Concept: Temperature & Creativity

Many AI tools have a "temperature" or "creativity" setting (often 0–1 or 0–2):

  • Low temperature (0–0.3): Predictable, consistent, factual—like a careful researcher
  • Medium temperature (0.7–0.8): Balanced—good for most writing tasks
  • High temperature (1.0+): Unpredictable, creative, varied—like a brainstorming session

Note: Not all interfaces expose this setting, but understanding it helps you frame prompts!

How to Use This in Real Work

  • If accuracy matters: lower temperature + tighter constraints + ask for sources/assumptions.
  • If you need options: raise temperature + ask for variation + request multiple approaches.
  • If outputs are “samey”: increase temperature and explicitly ask for diverse formats.
  • If outputs are “wild”: decrease temperature and add structure (bullets, rubric, criteria).
